Der Inhalt für AWS Wolke Quiz wurde von rbashish entwickelt. AWS Wolke Quiz läuft jedoch auf Systemen, die von Google bereitgestellt werden, und rbashish hat keinen vollständigen Zugriff auf die App. Insbesondere hat rbashish keine Möglichkeit, auf die Nachrichten zuzugreifen, die Sie an das Amazon Web Services Quiz senden, oder auf die Antworten, die es an Sie zurücksendet.

rbashish kann nicht personenbezogene Daten von Google über die Verwendung von AWS Wolke Quiz erhalten. So kann er beispielsweise auf Nutzungsinformationen zugreifen, wie viele Benutzer die App verwenden, in welchen geografischen Regionen sie sich befinden, sowie auf Basisdaten wie Sprache, Gerätetyp und Länge und Häufigkeit der Anwendung. Keiner der Informationen, die er von Google erhält, identifiziert Sie und zeigt ihm auch nicht an, welche Informationen Sie an die App gesendet haben oder welche spezifischen Antworten die App Ihnen gesendet hat.

Wenn Sie AWS Wolke Quiz verwenden, verwenden Sie auch den Google-Assistenten. In der Google-Datenschutzrichtlinie wird beschrieben, wie Google Daten über Ihre Nutzung der App sammelt und verwendet. Möglicherweise möchten Sie in der Dokumentation von Google nachlesen, welche Informationen mit Ihrem Google Assistant geteilt werden und wie Sie Ihre Google Assistant-Aktivitäten löschen.

rbashish kann in Zukunft AWS Wolke Quiz so modifizieren, dass es auf einer Infrastruktur läuft, auf die DEVELOPER vollen Zugriff hat, wodurch DEVELOPER auf zusätzliche Informationen über Ihre Interaktion mit der App zugreifen kann. Sollte dies der Fall sein, wird DEVELOPER seinen Verpflichtungen nach geltendem Datenschutzrecht nachkommen, einschließlich der Aktualisierung dieser Datenschutzrichtlinie nach Bedarf.
